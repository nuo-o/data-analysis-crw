Remove outliers and Plot the distribution¶

We see that 10% of the dataset consist of null values. According to wikipedia, CS:go was released on August 21 2012. https://en.wikipedia.org/wiki/Counter-Strike:_Global_Offensive The dataset we are provided was retrieved on January 08 2018. Therefore we can assume that any values which indicate they have been playing longer than this period are false values. The value 169862400 seconds is the total time the game has existed to the public. Since some of the variable "time played" measured for some players are higher than this value, we can reason that the dataset has other values that could be false. However, we have no way to judge for these other values, hence we have kept them within the dataset we will analyse.
